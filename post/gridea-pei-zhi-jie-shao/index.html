Gridea配置介绍

这个自搭blog可以基于GitHub Page也可以Coding Page。后者腾讯平台不熟，想折腾去b站看教程(Gridea+Coding Page)。具体搭建流程总的来说就是下载应用平台，github建仓建Token，再二者互通一下就行具体见B站或者官方文稿。（有GitHub账号情况下，五分钟就好了）

在此比较重要的，给想新入坑的朋友介绍就是：

  1. GitHub page在本地Chrome预览ok，但实际加载不出来的情况。这涉及到浏览器中对该网页安全加载项的判断，调白名单就行（因此，我认为这个blog以目前这样并不适合给大家分享给其他PC端用户自己的一些记录，自己愿意弄调成白名单可以当自己跨平台笔记阅读用）。
  2. 移动端可以自由访问。

大家有更好PC阅读问题解决方法欢迎邮件私戳我！
